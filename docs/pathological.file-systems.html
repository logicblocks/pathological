pathological.file-systems

*file-system*

dynamic

The file system to use in various calls where an explicit file system is optional, e.g., pathological.paths/path, pathological.path-matchers/path-matcher, pathological.principals/->user-principal and pathological.principals/->group-principal

By default, the default-file-system is used. However this var is dynamic and can be altered / rebound to any file system instance.
